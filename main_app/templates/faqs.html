Frequently Asked Questions

When I add the event to my account, does it add it to my Google Calendar?

Currently Volun-Go does not link with Google Calendar. However, it is a feature we are working on, so check back!

I don't see any volunteer opportunities that I like, can I create one?

Absolutely! Just click on the Add Event link in the navigation bar.

I created an event but I put the wrong date. Is there a way to edit?

Events can only be edited or deleted by the event creator. If you created an event, simply go to the event detail page to find the Edit and Delete buttons.
